The website was developed under the U.S. Department of Agriculture Grant # 2017-68005-26867. It is colloquially known as the Sustainable Bioeconomy for Arid Regions (SBAR) project. This project aims to assess the economic and environmentally viability of two bio-product crops: Guayule & Guar. The project combines engineering, science, and outreach practices. Two goals of the SBAR project are to:

• Develop high-throughput phenotyping to support crop expansion using remote-sensing methods to create interactive databases/tools

• Deploy agronomic production practices; identify agronomic information for salinity, herbicide, and nutrients to support production; provide irrigation apps using algorithms to growers.

This website hopes to accomplish both of those goals by combining databases with computer model analysis. This website allows users to view and download model data, predict irrigation events, and input data into the model for analysis. The WINDS model is a python3-based model that predicts various soil & water parameters for crops. It also features an interactive irrigation calculator that can predict when the next irrigation event for a particular planting will occur. Users can also input their own planting data that will be used by the model.
