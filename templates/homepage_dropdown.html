Welcome to the WINDS website!

This website features a wide array of tools to allow you to interact with the WINDS model. The tabs in the navigation bar are multi-purpose: Hover over them to see the various options at your disposal.


Data Viewers Tab

The data viewers allow you to interact with model outputs and predict future irrigation.


Data Entry Tab

The data entry allows registered users to establish their own plantings, irrigation data, neutron probe data, and remote sensing data. These inputs are then used by the model to produce outputs.
